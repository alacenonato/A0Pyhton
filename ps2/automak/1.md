O Open PS2 Loader (OPL) na versão 1.3.4.2, por si só, não executa jogos de PS1 diretamente, pois ele é um "Open PS2 Loader" . Para rodar esses jogos, você precisará usar um emulador separado chamado **POPStarter** e configurá-lo manualmente dentro do OPL .

Aqui está um guia passo a passo para fazer essa configuração funcionar:

### 📥 Passo 1: Baixe os Arquivos Necessários
Você precisará de quatro componentes principais, todos disponíveis gratuitamente online:
1.  **POPStarter**: O arquivo principal do emulador (`POPSTARTER.ELF`).
2.  **POPS_IOX.PAK**: Um arquivo essencial que contém o código de emulação (não pode ser redistriburado, mas é fácil de encontrar com o MD5 `a625d0b3036823cdbf04a3c0e1648901`).
3.  **CUE2POPS**: Uma ferramenta de PC para converter seus jogos de PS1 (no formato `.bin/.cue`) para o formato `.VCD`, que é lido pelo POPStarter.
4.  **OPL Manager (Opcional)**: Um programa de PC que facilita muito a organização e transferência dos arquivos.

### 🛠️ Passo 2: Prepare os Jogos de PS1 no PC
1.  **Converta o jogo**: Pegue o arquivo `.cue` do seu jogo de PS1 e arraste-o sobre o executável do `CUE2POPS.exe`. Isso vai gerar um arquivo com a extensão `.VCD` .
2.  **Organize os arquivos**: A forma de organizar os arquivos muda dependendo de onde você vai armazenar os jogos (USB, HDD interno ou rede/SMB).
    *   **Para USB**: Na raiz do seu pendrive, crie uma pasta chamada `POPS`. Coloque todos os seus arquivos `.VCD` dentro dela.
    *   **Para HDD Interno ou Rede (SMB)**: Dentro da pasta do OPL (geralmente `OPL/`), crie uma subpasta chamada `POPS` e coloque os arquivos `.VCD` lá .

### ⚙️ Passo 3: Configure o Emulador POPStarter
Esta é a etapa mais importante para que o OPL reconheça os jogos.
1.  **Encontre o `POPS_IOX.PAK`** e coloque-o na mesma pasta `POPS` onde estão os seus jogos `.VCD` .
2.  **Crie um "lançador" para cada jogo**: Você precisa de um arquivo `.ELF` para cada jogo. A maneira mais fácil é:
    *   Copie o arquivo `POPSTARTER.ELF` para a pasta `POPS`.
    *   Renomeie essa cópia para ter o **mesmo nome do seu arquivo `.VCD`**, mas com a extensão `.ELF`. Adicione o prefixo `XX.` no início.
    *   **Exemplo**: Se seu jogo é `Crash Bandicoot.VCD`, o lançador deve se chamar `XX.Crash Bandicoot.ELF`.
3.  **Resultado final**: Dentro da pasta `POPS`, você terá algo assim:
    *   `POPS_IOX.PAK`
    *   `XX.MeuJogo.ELF` (o lançador)
    *   `MeuJogo.VCD` (o jogo convertido)

### 🕹️ Passo 4: Execute os Jogos no Seu PS2
Com os arquivos no lugar, você tem duas maneiras de rodar os jogos pelo OPL:

*   **Método 1: Pelo Menu de Aplicativos (Recomendado)**
    É a forma mais limpa e funciona com versões oficiais do OPL.
    1.  No seu dispositivo (USB, HDD), crie uma pasta chamada `APPS`.
    2.  Dentro de `APPS`, crie uma pasta com o nome do seu jogo (ex: `Crash Bandicoot`).
    3.  Coloque o lançador (o arquivo `XX.Crash Bandicoot.ELF`) dentro dela.
    4.  Crie um arquivo de texto chamado `title.cfg` dentro desta mesma pasta com o seguinte conteúdo :
        ```
        title=Crash Bandicoot
        boot=XX.Crash Bandicoot.ELF
        ```
    5.  No OPL, vá até a aba **"APPS"**. Seu jogo deve aparecer na lista. Basta selecioná-lo para iniciar.

*   **Método 2: Via Navegador de Arquivos (uLaunchELF)**
    É uma forma mais direta.
    1.  No seu PS2, abra o **uLaunchELF**.
    2.  Navegue até a pasta `POPS` onde você colocou os arquivos.
    3.  Selecione o arquivo `XX.MeuJogo.ELF` e aperte `OK`. O jogo vai começar.

### 💡 Dicas Importantes
*   **Performance**: Diferente dos jogos de PS2, os jogos de PS1 rodando via POPStarter por **USB costumam ter um desempenho muito bom, sem engasgos** nas cenas em vídeo. Isso porque a exigência de leitura de dados é menor .
*   **Alimentação USB**: Se for usar um **HD externo de 2.5"** (aquele portátil, sem fonte de energia própria) na porta USB do PS2, ele pode não receber energia suficiente para funcionar. Nesse caso, o ideal é usar um **cabo Y USB** (que conecta em duas portas do PS2 ao mesmo tempo) ou um **pen drive** comum .
*   **Qual Versão do OPL Usar?**: A versão 1.3.4.2 que você tem é uma versão oficial recente e funciona muito bem para PS2. Para rodar PS1, você **não precisa** de uma versão modificada, basta seguir o guia do "Menu de Aplicativos" acima. Versões antigas com uma aba "PS1" dedicada não são mais recomendadas .
