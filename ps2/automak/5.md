Como você está no Linux (pelo prompt `alace@alace-RV411`), vou te passar os comandos para renomear todos os arquivos `.ELF` de uma vez.

---

## 🐧 Comandos para renomear no Linux

Abra um terminal e digite os comandos abaixo:

### 1. Primeiro, entre na pasta correta:
```bash
cd "/media/alace/My Passport/2POPS"
```

### 2. Renomeie todos os arquivos `XX.` para `SB.` de uma vez:
```bash
for arquivo in XX.*.ELF; do
    novo_nome=$(echo "$arquivo" | sed 's/^XX\./SB./')
    mv "$arquivo" "$novo_nome"
    echo "Renomeado: $arquivo -> $novo_nome"
done
```

### 3. Verifique se funcionou:
```bash
ls -la *.ELF
```

Agora você deve ver todos os arquivos começando com `SB.` em vez de `XX.`

## ⚠️ Atenção: Arquivos com nome diferente

Notei que alguns arquivos `.ELF` têm nome um pouco diferente do `.VCD`. Exemplo:

```
XX.SLPS_022.55.Rockman 2 - Dr.ELF   ← ELF
SLPS_022.55.Rockman 2 - Dr.VCD      ← VCD (nome ligeiramente diferente)
```

Para funcionar, o **nome do ELF** e do **VCD** precisam ser **idênticos** (exceto a extensão).

Depois de renomear de `XX.` para `SB.`, verifique se os nomes batem. Se não baterem, me avise que te ajudo a corrigir.
