do bem, não se preocupe em não saber o endereço agora. O importante é que sua estrutura de pastas (`DVD` e `CD`) já está correta para os jogos de PS2. Agora vamos adicionar o suporte para PS1.

Como você usa **SMB (rede)** , a estrutura é um pouco diferente do USB. Vamos organizar passo a passo.

---

## 📂 Estrutura de Pastas para PS1 via SMB

Na mesma pasta compartilhada onde você tem as pastas `DVD` e `CD`, você precisará criar/ajustar o seguinte:

```
Pasta Compartilhada (ex: PS2SMB)
│
├── 📁 DVD          (jogos PS2 em DVD)
├── 📁 CD           (jogos PS2 em CD)
├── 📁 POPS         (arquivos dos jogos PS1 e emulador)
│   ├── SB.NomeDoJogo.ELF   (lançador)
│   ├── NomeDoJogo.VCD      (jogo convertido)
│   └── POPS_IOX.PAK        (arquivo do emulador)
│
├── 📁 APPS         (para listar os jogos no menu do OPL)
│   └── 📁 NomeDoJogo
│       ├── title.cfg
│       └── SB.NomeDoJogo.ELF
│
└── 📁 CFG          (configurações, opcional)
```

---

## 🔧 Passo 1: Criar a pasta POPS

Na **raiz da sua pasta compartilhada** (mesmo local onde ficam `DVD` e `CD`), crie uma pasta chamada `POPS`.

---

## 📥 Passo 2: Colocar os arquivos do emulador

Dentro da pasta `POPS`, coloque:

1. **`POPS_IOX.PAK`** - Arquivo essencial do emulador (procure online, MD5: `a625d0b3036823cdbf04a3c0e1648901`) 
2. **`POPSTARTER.ELF`** - O executável do emulador

> ⚠️ **Importante para SMB**: Diferente do USB que usa o prefixo `XX.`, no SMB você deve usar o prefixo **`SB.`** (de SMB) 

---

## 🎮 Passo 3: Preparar cada jogo

Para cada jogo de PS1 que você quiser rodar:

1. **Converter para `.VCD`**:
   - Pegue o arquivo `.cue` do seu jogo
   - Arraste sobre o `CUE2POPS.exe`
   - Será gerado um arquivo `.VCD`

2. **Criar o lançador**:
   - Copie o arquivo `POPSTARTER.ELF`
   - Renomeie a cópia para: **`SB.NomeDoJogo.ELF`**
   - Exemplo: `SB.Crash Bandicoot.ELF`

3. **Organizar na pasta POPS**:
   ```
   POPS/
   ├── SB.Crash Bandicoot.ELF
   ├── Crash Bandicoot.VCD
   └── POPS_IOX.PAK
   ```

---

## 🖥️ Passo 4: Fazer os jogos aparecerem no OPL (Método APPS)

Para que os jogos apareçam na lista do OPL, você precisa criar uma estrutura na pasta `APPS`:

1. **Na raiz da pasta compartilhada**, crie (ou verifique se já existe) a pasta `APPS`

2. **Dentro de `APPS`**, crie uma pasta para cada jogo:
   ```
   APPS/
   └── Crash Bandicoot/
       ├── title.cfg
       └── SB.Crash Bandicoot.ELF
   ```

3. **Crie o arquivo `title.cfg`** com o seguinte conteúdo :
   ```
   title=Crash Bandicoot
   boot=SB.Crash Bandicoot.ELF
   ```

4. **Copie o arquivo `SB.Crash Bandicoot.ELF`** para dentro desta pasta

---

## ⚙️ Passo 5: Configurar no OPL

1. **Ligue o PS2 e abra o OPL**
2. Vá para **Settings** → **Apps**
3. Certifique-se de que:
   - `Show APPS` está como `On`
S List Path` está configurado (padrão é `APPS/`)
4. Salve as configurações
5. Volte ao menu principal e vá para a aba **APPS**

Seus jogos de PS1 devem aparecer listados! 

---

## 🐛 Se não funcionar: Configuração via `conf_apps.cfg`

Algumas versões do OPL precisam de um arquivo de configuração adicional. Na **raiz da pasta compartilhada**, crie um arquivo chamado `conf_apps.cfg` com o seguinte conteúdo :

```
Crash Bandicoot=smb0:/APPS/Crash Bandicoot/SB.Crash Bandicoot.ELF
```

Para cada jogo, adicione uma linha.
