## ✅ Solução 1: Configurar via aba APPS (recomendado)

Para versões oficiais do OPL, o método mais confiável é usar a aba **APPS** em vez da aba "PS1 Games" .

### Passo 1: Verifique a estrutura APPS

Entre na pasta compartilhada no Linux e execute:

```bash
cd "/media/alace/My Passport"
ls -la APPS/
```

Se não houver pastas, crie a estrutura correta:

```bash
cd "/media/alace/My Passport/POPS"

# Para cada jogo, crie a estrutura APPS
for elf in SB.*.ELF; do
    # Extrai o nome do jogo
    nome_base=$(echo "$elf" | sed 's/^SB\.//' | sed 's/\.ELF$//')
    nome_jogo=$(echo "$nome_base" | cut -d'.' -f3- | sed 's/_/ /g')
    
    # Cria pasta em APPS
    mkdir -p "../APPS/$nome_jogo"
    
    # Copia o ELF
    cp "$elf" "../APPS/$nome_jogo/"
    
    # Cria title.cfg
    cat > "../APPS/$nome_jogo/title.cfg" << EOF
title=$nome_jogo
boot=$elf
EOF
    echo "Configurado: $nome_jogo"
done
```

### Passo 2: Configure o OPL

No PS2:
1. Entre em **Settings** → **Apps**
2. Ative: `Show APPS` = `On`
3. `APPS List Path` = `APPS/`
4. Salve e volte ao menu principal
5. Acesse a aba **APPS** (não "PS1 Games")

## 🔧 Solução 2: Verificar arquivos do emulador

O POPStarter pode travar no loading se algum arquivo essencial estiver corrompido ou com MD5 incorreto .

### Verifique o MD5 do POPS_IOX.PAK:

```bash
cd "/media/alace/My Passport/POPS"
md5sum POPS_IOX.PAK
```

O resultado **deve ser**: `a625d0b3036823cdbf04a3c0e1648901` 

Se for diferente, você precisa baixar uma versão correta.

### Verifique os outros arquivos:

```bash
md5sum POPS.ELF
# Deve ser: 355a892a8ce4e4a105469d4ef6f39a42

md5sum IOPRP252.IMG
# Deve ser: 1db9c6020a2cd445a7bb176a1a3dd418
```

## ⚠️ Solução 3: Remover arquivo de configuração conflitante

Às vezes, o POPStarter encontra um arquivo `conf_elm.cfg` ou `conf_elmz.cfg` que faz com que ele retorne ao menu do PS2 em vez de iniciar o jogo .

```bash
cd "/media/alace/My Passport"
find . -name "conf_elm*.cfg" -type f
```

Se encontrar algum, remova:

```bash
rm -f ./conf_elm*.cfg
```

## 🔄 Solução 4: Testar com versão alternativa do POPStarter

Sua versão do `POPSTARTER.ELF` é de 2019 (pelo `ls -la`). Pode ser que uma versão mais recente resolva.

Baixe a versão mais recente do POPStarter (r13 WIP 06 ou superior) e substitua :

```bash
# Baixe o POPStarter mais recente (você precisa encontrar o link)
# Depois de baixado:
cp ~/Downloads/POPSTARTER.ELF "/media/alace/My Passport/POPS/"
chmod +x "/media/alace/My Passport/POPS/POPSTARTER.ELF"

# Recrie os ELFs dos jogos
cd "/media/alace/My Passport/POPS"
for vcd in *.VCD; do
    nome_base="${vcd%.VCD}"
    cp POPSTARTER.ELF "SB.${nome_base}.ELF"
    echo "Recriado: SB.${nome_base}.ELF"
done
```
