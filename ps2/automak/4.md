## ❌ O problema principal

Olhando seus arquivos `.ELF`, eles estão com o prefixo **`XX.`** no nome.

Exemplo:
- `XX.SLUS_011.79.ValkyrieCD2.ELF`

**Para SMB (rede), o prefixo correto é `SB.` , não `XX.`**

O prefixo `XX.` é para **USB**. Para **rede (SMB)** , você precisa usar **`SB.`**

---

## 🔧 O que você precisa fazer

### Para cada jogo que você quer rodar:

1. **Renomeie o arquivo `.ELF`** de `XX.NomeDoJogo.ELF` para `SB.NomeDoJogo.ELF`

2. **Mantenha o arquivo `.VCD` com o mesmo nome** (sem o prefixo)

Exemplo para o jogo Tarzan:
```
Antes:
  XX.SCUS_944.56.Tarzan (Dublado).ELF
  SCUS_944.56.Tarzan (Dublado).VCD

Depois:
  SB.SCUS_944.56.Tarzan (Dublado).ELF
  SCUS_944.56.Tarzan (Dublado).VCD
```
