## 🔧 Passo a passo para organizar

### 1. Mova os arquivos .VCD e .ELF para a pasta POPS (se já não estiverem)

Seus arquivos já estão em `/2POPS`. Você precisa que eles estejam na pasta que o PS2 acessa via rede.

```bash
cd "/media/alace/My Passport"
```

Se sua pasta compartilhada for a raiz do HD (`My Passport`), mova os arquivos:

```bash
# Garanta que a pasta POPS existe na raiz
mkdir -p "POPS"

# Copie os jogos para dentro de POPS
cp /media/alace/My\ Passport/2POPS/*.VCD /media/alace/My\ Passport/POPS/
cp /media/alace/My\ Passport/2POPS/SB.*.ELF /media/alace/My\ Passport/POPS/

# Copie também o arquivo essencial POPS_IOX.PAK se ainda não estiver lá
cp /media/alace/My\ Passport/POPS/POPS_IOX.PAK /media/alace/My\ Passport/POPS/ 2>/dev/null || echo "POPS_IOX.PAK já deve estar na pasta POPS"
```

### 2. Crie a estrutura APPS para o OPL enxergar

```bash
# Crie a pasta APPS na raiz
mkdir -p "APPS"

# Agora, para cada jogo, crie uma pasta com title.cfg
# Vou fazer um loop para todos os seus jogos
cd "/media/alace/My Passport/POPS"

for elf in SB.*.ELF; do
    # Pega o nome base sem SB. e sem .ELF
    nome_base=$(echo "$elf" | sed 's/^SB\.//' | sed 's/\.ELF$//')
    
    # Remove a extensão .VCD do nome se tiver (ex: SCUS_941.94.Gran Turismo)
    nome_jogo=$(echo "$nome_base" | sed 's/\.VCD$//')
    
    # Cria a pasta do jogo em APPS
    mkdir -p "../../APPS/$nome_jogo"
    
    # Copia o ELF para dentro da pasta
    cp "$elf" "../../APPS/$nome_jogo/"
    
    # Cria o arquivo title.cfg
    cat > "../../APPS/$nome_jogo/title.cfg" << EOF
title=$nome_jogo
boot=$elf
EOF
    
    echo "Criado: APPS/$nome_jogo/"
done
```

### 3. Verifique se o POPS_IOX.PAK está na pasta POPS

```bash
ls -la /media/alace/My\ Passport/POPS/POPS_IOX.PAK
```

Se não estiver, copie de onde você tem:

```bash
# Se estava na pasta 2POPS
cp /media/alace/My\ Passport/2POPS/POPS_IOX.PAK /media/alace/My\ Passport/POPS/
```

## 🎮 No Playstation 2 (OPL)

### Configuração no OPL:

1. **Abra o OPL** no PS2
2. Vá para **Settings** → **Apps**
3. Ative: `Show APPS` = `On`
4. Mantenha: `APPS List Path` = `APPS/` (padrão)
5. Salve as configurações (aperte **OK** e depois **Save Changes**)

### Para jogar:

1. Volte ao menu principal do OPL
2. Vá para a aba **APPS** (não a PS1, não a HDD Games - é APPS)
3. Seus jogos devem aparecer listados pelo nome que você colocou no `title.cfg`
4. Selecione um jogo e aperte **X** para iniciar

## ⚠️ Se os jogos não aparecerem

### Solução 1: Arquivo conf_apps.cfg

Na raiz da pasta compartilhada, crie um arquivo `conf_apps.cfg`:

```bash
cd "/media/alace/My Passport"
cat > conf_apps.cfg << 'EOF'
Gran Turismo=smb0:/APPS/Gran Turismo/SB.SCUS_941.94.Gran Turismo.ELF
Bloody Roar=smb0:/APPS/Bloody Roar/SB.SCUS_941.99.Bloody Roar.ELF
EOF
```

Adicione uma linha para cada jogo.

### Solução 2: Verifique o nome dos arquivos

O nome do `.ELF` dentro da pasta `APPS/NomeDoJogo/` deve ser **exatamente igual** ao nome do `.VCD` na pasta `POPS/`, exceto a extensão.

Exemplo correto:
- `POPS/SCUS_941.94.Gran Turismo.VCD`
- `APPS/Gran Turismo/SB.SCUS_941.94.Gran Turismo.ELF`
