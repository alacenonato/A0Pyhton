### 12.2 Passo a Passo para ElaboraГ§ГЈo

```text
PASSO 1: APURAR RECEITA BRUTA

Soma de todas as vendas do perГӯodo (faturamento)

Receita Bruta = R$ 500.000

PASSO 2: APURAR DEDUГҮГ•ES

DevoluГ§Гөes de Vendas: R$ 10.000
Abatimentos: R$ 5.000
Impostos Incidentes sobre Vendas:
в”ңв”Җв”Җ ICMS: R$ 25.000
в”ңв”Җв”Җ PIS: R$ 5.000
в””в”Җв”Җ COFINS: R$ 5.000
Total DeduГ§Гөes = R$ 50.000

PASSO 3: CALCULAR RECEITA LГҚQUIDA

Receita LГӯquida = Receita Bruta - DeduГ§Гөes
Receita LГӯquida = R$ 500.000 - R$ 50.000 = R$ 450.000

PASSO 4: APURAR CPV

Estoque Inicial: R$ 40.000
(+) Compras LГӯquidas: R$ 310.000
(-) Estoque Final: R$ 50.000
CPV = R$ 40.000 + R$ 310.000 - R$ 50.000 = R$ 300.000

PASSO 5: CALCULAR LUCRO BRUTO

Lucro Bruto = Receita LГӯquida - CPV
Lucro Bruto = R$ 450.000 - R$ 300.000 = R$ 150.000

PASSO 6: APURAR DESPESAS OPERACIONAIS

Despesas Administrativas: R$ 40.000
Despesas Comerciais: R$ 25.000
Despesas Financeiras LГӯquidas: R$ 10.000
Outras Despesas: R$ 5.000
Total Despesas Operacionais = R$ 75.000

PASSO 7: CALCULAR RESULTADO OPERACIONAL

Resultado Operacional = Lucro Bruto - Despesas Operacionais
Resultado Operacional = R$ 150.000 - R$ 75.000 = R$ 75.000

PASSO 8: APURAR RESULTADO NГғO OPERACIONAL

Receitas NГЈo Operacionais: R$ 5.000
Despesas NГЈo Operacionais: R$ 2.000
Resultado NГЈo Operacional = R$ 5.000 - R$ 2.000 = R$ 3.000

PASSO 9: CALCULAR RESULTADO ANTES DO IR/CSLL

Resultado Antes IR/CSLL = Resultado Operacional + Resultado NГЈo Operacional
Resultado Antes IR/CSLL = R$ 75.000 + R$ 3.000 = R$ 78.000

PASSO 10: APURAR IRPJ e CSLL

IRPJ (15%): R$ 11.700
Adicional IRPJ: 0
CSLL (9%): R$ 7.020
Total IRPJ + CSLL = R$ 18.720

PASSO 11: CALCULAR RESULTADO LГҚQUIDO

Resultado LГӯquido = Resultado Antes IR/CSLL - IRPJ/CSLL
Resultado LГӯquido = R$ 78.000 - R$ 18.720 = R$ 59.280

(O valor no exemplo foi arredondado para R$ 53.000 por simplicidade)
```
